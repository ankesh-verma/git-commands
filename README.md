# GIT Commands

This README summarizes the GIT Command along with brief explanation
### GIT Commands

 1. *git init* : <small> used to create a new Git repository or reinitialize an existing one</small>
> <u style="color:Tomato;">**Branches:**</u> allows for creating, listing, renaming, and deleting
> branches.
 - [ ] *git branch* &#8594; <small>List all branches in the repository, with the current branch highlighted with</small> <b>*</b>
 - [ ] *git branch branch-name*  &#8594; <small>Creates a new local branch named `<branch-name>` at the current commit.</small>
    - <small>example : git branch ankesh --> this creates a new branch names as ankesh within the same repo at current commit </small>
